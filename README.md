# Mercy Paints 🎨
### A Division of A.L.O.I Chemicals

A premium commercial paint showroom and material procurement web application built for Mercy Paints — a real architectural coatings brand manufactured in Lagos, Nigeria.

## 🌐 Live Site
[mercypaint.vercel.app](https://mercypaint.vercel.app)

## 📌 What It Does
- Browse 57 signature paint colors with a live room visualizer
- Calculate exact material requirements for multiple rooms or wall sections
- Smart container optimization (4L Gallons vs 20L Drums)
- Automatic wastage guard deduction for doors and windows
- Logistics delivery cost calculator across Lagos and nationwide
- Dispatch full project quotes directly to WhatsApp or Email

## 🛠️ Built With
- Semantic HTML5
- Custom CSS3 with Glassmorphic Design
- Vanilla JavaScript (OOP Class Architecture)
